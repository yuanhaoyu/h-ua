# h-ua

According the UA to judge the current environment
 
## Quick start

<code>clone /src/h-ua.js </code> simple start 

```javascript
const enviorment = window.HAPPY_UA();
```

or <code>npm install happy-ua --save </code> use es6 modules

```javascript
import ua from 'happy-ua';
const enviorment = ua();
```
